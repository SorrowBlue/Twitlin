The name for the list. A list's name must start with a letter and can consist only of 25 or fewer letters, numbers, "-", or "_" characters.

Whether your list is public or private. Values can be UserList.Mode.PUBLIC or UserList.Mode.PRIVATE . If no mode is specified the list will be UserList.Mode.PUBLIC.
