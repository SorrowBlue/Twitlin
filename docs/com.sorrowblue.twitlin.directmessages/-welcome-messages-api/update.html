update

abstract suspend fun update(id: String, text: String, mediaId: String? = null, quickReply: QuickReply? = null): Response<WelcomeMessageData>
Content copied to clipboard

Updates a Welcome Message by the given ID. Updates to the WelcomeMessage are atomic. For example, if the Welcome Message currently has MessageData.quick_reply defined and you only provde text, the MessageData.quick_reply object will be removed from the Welcome Message.

Parameters

id

The id of the Welcome Message that should be updated.

text

The text of your Direct Message. URL encode as necessary. Max length of 10,000 characters. Max length of 9,990 characters if used as a Welcome Message.

mediaId

A media id to associate with the message. A Direct Message may only reference a single MediaResult.mediaId. See Uploading Media for further details on uploading media.

quickReply

The Quick Reply type to present to the user.
