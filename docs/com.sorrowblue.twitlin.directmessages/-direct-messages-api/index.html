Sending message events

To send a new Direct Message use DirectMessagesApi.new (message_create). Optionally, you may also attach Quick Replies or media (image, GIF or video).

Receiving messages events

  • You can retrieve Direct Messages from up to the past 30 days with DirectMessagesApi.list.

  • Consuming Direct Messages in real-time can be accomplished via webhooks with the Account Activity API.
