Tweets are the basic atomic building block of all things Twitter. Tweets are also known as “status updates.” The Tweet object has a long list of ‘root-level’ attributes, including fundamental attributes such as id, createdAt, and text. Tweet objects are also the ‘parent’ object to several child objects. Tweet child objects include user, entities, and extendedEntities. Tweets that are geo-tagged will have a place child object.
