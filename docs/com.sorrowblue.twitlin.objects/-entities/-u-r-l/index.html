data class URL(url: String, expandedUrl: String, displayUrl: String, indices: List<Int>, unwound: Entities.URL.Unwound?) : JvmSerializable
Content copied to clipboard

The Tweet.entities section will contain a Entities.urls array containing an object for every link included in the Tweet body, and include an empty array if no links are present. The has:links Operator will match if there is at least one item in the array. The url: Operator is used to match on the expandedUrl attribute. If you are using the Expanded URL enrichment, the url: Operator is used to match on the Unwound.url (fully unwound URL) attribute. If you are using the Expanded URL enrichment, the url_title: and url_decription: Operators are used to match on the Unwound.title and Unwound.description attributes.
