addEntries

abstract suspend fun addEntries(id: String, tweetId: String, relativeTo: String? = null, above: Boolean = true): Response<Collections<Unit, CollectionResponse.Errors>>
Content copied to clipboard

Add a specified Tweet to a Collection.

A collection will store up to a few thousand Tweets. Adding a Tweet to a collection beyond its allowed capacity will remove the oldest Tweet in the collection based on the time it was added to the collection.

Use CollectionsApi.curateEntries to add Tweets to a Collection in bulk.

The identifier of the Tweet used for relative positioning in a TimelineOrder.CURATION_REVERSE_CHRON ordered collection.

Set to false to insert the specified tweetId below the relativeTo Tweet in the collection. Default: true
