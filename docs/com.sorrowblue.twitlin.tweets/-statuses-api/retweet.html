retweet

abstract suspend fun retweet(id: String, trimUser: Boolean = false): Response<Tweet>
Content copied to clipboard

Retweets a tweet. Returns the original Tweet with Retweet details embedded.

Usage Notes:

  • This method is subject to update limits. A HTTP 403 will be returned if this limit as been hit.

  • Twitter will ignore attempts to perform duplicate retweets.

  • The Tweet.retweetCount will be current as of when the payload is generated and may not reflect the exact count. It is intended as an approximation.
