data class Hashtag(indices: List<Int>, text: String) : JvmSerializable
Content copied to clipboard

The TwitterTweet.entities section will contain a Entities.hashtags array containing an object for every hashtag included in the Tweet body, and include an empty array if no hashtags are present. The PowerTrack # Operator is used to match on the text attribute. The has:hashtags Operator will match if there is at least one item in the array.
