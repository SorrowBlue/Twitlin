abstract suspend fun members(listId: Long, count: Int? = null, cursor: Long? = null, includeEntities: Boolean? = null, skipStatus: Boolean? = null): Response<PagingTwitterUser>
Content copied to clipboard

Returns the members of the specified list. Private list members will only be shown if the authenticated user owns the specified list.

Return

Paged Twitter User

Parameters

listId

The numerical TwitterList.id of the list.

count

Specifies the number of results to return per page (see cursor below). The default is 20, with a maximum of 5,000.

cursor

Causes the collection of list members to be broken into "pages" of consistent sizes (specified by the count parameter). If no cursor is provided, a value of -1 will be assumed, which is the first "page".

includeEntities

The com.sorrowblue.twitlin.objects.TwitterUser node will not be included when set to false.

skipStatus

When set to either true statuses will not be included in the returned user objects.
